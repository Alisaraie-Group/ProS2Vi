The PDB code may be invalid or the RCSB server may be temporarily unavailable.

Invalid File Format

The file must be a standard PDB (.pdb) or mmCIF (.cif) file with proper formatting.

High DPI Requires Poppler

DPI settings above 100 require poppler-utils to be installed on the system.
