Image Size Limit Exceeded

The generated image is too large to process safely.

Why This Happens

This error occurs when the combination of structure size and DPI setting produces an image that exceeds memory limits. Large protein structures with many residues at high DPI settings can generate extremely large images.

Solutions
Lower the DPI Setting

Try using 100 DPI for preview or 300 DPI for publication instead of higher values.

Increase Residues per Line

More residues per line means fewer rows, resulting in a smaller image height.

Use PDF Output

PDF files are vector-based and don't have the same size limitations. You can convert to image later.

Try Again with Different Settings
